Post API Documentation

Note: All endpoints require a valid JWT token in the Authorization header as "Bearer <token>" for authentication.

Get All Posts

GET /v1/post/

Fetches all posts from the database.

Request Header:
Authorization: Bearer <token>

Response:
Status: 200 OK
[
    {
        "_id": "postId",
        "text": "Post content",
        "author": "userId",
        "category": "categoryName",
        "createdAt": "2024-09-09T10:00:00.000Z",
        "updatedAt": "2024-09-09T10:00:00.000Z"
    },
    ...
]

Create Post

POST /v1/post/

Creates a new post.

Request Header:
Authorization: Bearer <token>

Request Body:
{
    "text": "Post content",
    "category": "Category name"
}

Response:
Status: 201 Created
{
    "_id": "postId"
}

Update Post

PUT /v1/post/

Updates an existing post.

Request Header:
Authorization: Bearer <token>

Request Body:
{
    "postId": "postId",
    "text": "Updated post content",
    "category": "Updated category"
}

Response:
Status: 200 OK
{
    "_id": "postId"
}

Delete Post

DELETE /v1/post/

Deletes a post.

Request Header:
Authorization: Bearer <token>

Request Body:
{
    "postId": "postId"
}

Response:
Status: 200 OK
{
    "_id": "deletedPostId"
}
