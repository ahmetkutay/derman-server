API Documentation

User API

GET /api/user/{id}

Retrieve user details by ID.

Parameters:

  • id (integer) - The ID of the user.

Response:

{
    "id": 1,
    "name": "John Doe",
    "email": "john@example.com"
}

POST /api/user

Create a new user.

Body Parameters:

  • name (string) - The name of the user.
  • email (string) - The email of the user.
  • password (string) - The password of the user.

Response:

{
    "id": 2,
    "name": "Jane Doe",
    "email": "jane@example.com"
}

Auth API

POST /api/auth/login

Authenticate a user and return a token.

Body Parameters:

  • email (string) - The email of the user.
  • password (string) - The password of the user.

Response:

{
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
}

POST /api/auth/register

Register a new user.

Body Parameters:

  • name (string) - The name of the user.
  • email (string) - The email of the user.
  • password (string) - The password of the user.

Response:

{
    "id": 3,
    "name": "Alice Doe",
    "email": "alice@example.com",
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
}

Post API

GET /api/post/{id}

Retrieve a post by ID.

Parameters:

  • id (integer) - The ID of the post.

Response:

{
    "id": 1,
    "title": "My first post",
    "content": "This is the content of the post."
}

POST /api/post

Create a new post.

Body Parameters:

  • title (string) - The title of the post.
  • content (string) - The content of the post.

Response:

{
    "id": 2,
    "title": "Another post",
    "content": "This is another post content."
}

Chat API

GET /api/chat/{id}

Retrieve chat messages by chat ID.

Parameters:

  • id (integer) - The ID of the chat.

Response:

[
    {
        "sender": "John Doe",
        "message": "Hello, how are you?",
        "timestamp": "2024-08-30T10:00:00Z"
    },
    {
        "sender": "Jane Doe",
        "message": "I'm good, thanks!",
        "timestamp": "2024-08-30T10:01:00Z"
    }
]

POST /api/chat

Send a new chat message.

Body Parameters:

  • chat_id (integer) - The ID of the chat.
  • sender (string) - The name of the sender.
  • message (string) - The message content.

Response:

{
    "chat_id": 1,
    "sender": "John Doe",
    "message": "See you soon!",
    "timestamp": "2024-08-30T10:02:00Z"
}
