Auth API Documentation

Login

POST /v1/auth/

Request Body:

{
    "username": "user123",
    "password": "password123"
}

Response:

{
    "accessToken": "yourAccessToken",
    "refreshedToken": "yourRefreshToken"
}

Register

POST /v1/auth/register

Request Body:

{
    "username": "user123",
    "first_name": "John",
    "last_name": "Doe",
    "email": "john.doe@example.com",
    "password": "password123",
    "phone": "1234567890",
    "birthdate": "1990-01-01",
    "gender": "male"
}

Response:

{
    "user": "userId"
}

Logout

POST /v1/auth/logout

Request Header:

Authorization: Bearer <token>

Response:

{
    "message": "Logged out successfully"
}

Verification

POST /v1/auth/verification

Request Body:

{
    "token": "verificationToken",
    "id": "userId"
}

Response:

"Verified"

Comment & Reply API Documentation

Create Comment

POST /v1/comment/:postId

Request Body:

{
    "text": "This is a comment"
}

Response:

{
    "_id": "commentId"
}

Update Comment

PUT /v1/comment/:postId/:commentId

Request Body:

{
    "text": "Updated comment text"
}

Response:

{
    "_id": "commentId"
}

Delete Comment

DELETE /v1/comment/:postId/:commentId

Response:

{
    "_id": "commentId"
}

Create Reply

POST /v1/comment/:postId/:commentId/reply

Request Body:

{
    "text": "This is a reply"
}

Response:

{
    "_id": "replyId"
}

Update Reply

PUT /v1/comment/:postId/:commentId/reply/:replyId

Request Body:

{
    "text": "Updated reply text"
}

Response:

{
    "_id": "replyId"
}

Delete Reply

DELETE /v1/comment/:postId/:commentId/reply/:replyId

Response:

{
    "_id": "replyId"
}

Post API Documentation

Note: All endpoints require a valid JWT token in the Authorization header as "Bearer <token>" for authentication.

Get All Posts

GET /v1/post/

Fetches all posts from the database.

Request Header:
Authorization: Bearer <token>

Response:
Status: 200 OK
[
    {
        "_id": "postId",
        "text": "Post content",
        "author": "userId",
        "category": "categoryName",
        "createdAt": "2024-09-09T10:00:00.000Z",
        "updatedAt": "2024-09-09T10:00:00.000Z"
    },
    ...
]

Create Post

POST /v1/post/

Creates a new post.

Request Header:
Authorization: Bearer <token>

Request Body:
{
    "text": "Post content",
    "category": "Category name"
}

Response:
Status: 201 Created
{
    "_id": "postId"
}

Update Post

PUT /v1/post/:postId

Updates an existing post.

Request Header:
Authorization: Bearer <token>

Request Body:
{
    "text": "Updated post content",
    "category": "Updated category name"
}

Response:
Status: 200 OK
{
    "_id": "postId"
}

Delete Post

DELETE /v1/post/:postId

Deletes a post by ID.

Request Header:
Authorization: Bearer <token>

Response:
Status: 200 OK
{
    "message": "Post deleted successfully"
}

User API Documentation

Get User Profile

GET /v1/user/:userId

Response:

{
    "id": "userId",
    "username": "user123",
    "email": "user@example.com",
    "firstName": "John",
    "lastName": "Doe",
    "phone": "1234567890"
}

Update User Profile

PUT /v1/user/:userId

Request Body:

{
    "username": "newUsername",
    "firstName": "John",
    "lastName": "Doe",
    "phone": "0987654321"
}

Response:

{
    "message": "Profile updated"
}

Delete User

DELETE /v1/user/:userId

Response:

{
    "message": "User deleted successfully"
}
