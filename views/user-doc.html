API Documentation

Note: All endpoints require a valid JWT token in the Authorization header as "Bearer <token>" for authentication.

User Routes

Request Password Reset Code

POST /v1/user/reset-password/code

Request Body:
{
    "_id": "userId"
}

Response:
Status: 200 OK
{
    "message": "Password reset code sent"
}

Reset Password

POST /v1/user/reset-password

Request Body:
{
    "_id": "userId",
    "token": "resetToken",
    "password": "oldPassword",
    "newPassword": "newPassword"
}

Response:
Status: 200 OK
{
    "message": "Password successfully reset"
}
