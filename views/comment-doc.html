Comment & Reply API Documentation

Create Comment

POST /v1/comment/:postId

Request Body:

{
    "text": "This is a comment"
}

Response:

{
    "_id": "commentId"
}

Update Comment

PUT /v1/comment/:postId/:commentId

Request Body:

{
    "text": "Updated comment text"
}

Response:

{
    "_id": "commentId"
}

Delete Comment

DELETE /v1/comment/:postId/:commentId

Response:

{
    "_id": "commentId"
}

Create Reply

POST /v1/comment/:postId/:commentId/reply

Request Body:

{
    "text": "This is a reply"
}

Response:

{
    "_id": "replyId"
}

Update Reply

PUT /v1/comment/:postId/:commentId/reply/:replyId

Request Body:

{
    "text": "Updated reply text"
}

Response:

{
    "_id": "replyId"
}

Delete Reply

DELETE /v1/comment/:postId/:commentId/reply/:replyId

Response:

{
    "_id": "replyId"
}
